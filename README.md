# LWRR : Landscape of Wheat Rust Resistance

> An information-rich landscape for wheat rust resistance loci facilitating design breeding for resistance improvement
A dynamic database with multiple functions for wheat design breeding of disease resistance

![image](https://github.com/user-attachments/assets/001a143d-ca0b-4658-b225-70496d467c4b)

## Introduction

LWRR employs a dynamic responsive architecture, comprising 7 main functional modules: "Population", "Sample", "Trait", "GWAS", "QTL", "Gene" and "Tools", facilitating rapid user navigation. With an intuitive online interface, LWRR provides breeders with rapid access to comprehensive information on wheat rust resistance loci, such as the distribution of disease resistance loci in a material and the differences in the utilization of different disease resistance loci. Users can also determine the number and identity of rust resistance loci present in specific accessions.
