                                                  1、项目简介

                                                  研究背景与意义

                                                  小麦是我国重要的粮食作物，小麦稳产事关我国粮食安全大局。在小麦生产中，条锈病危害巨大。小麦条锈菌可以通过气流远距离传播，造成条锈病大规模流行并导致小麦大幅减产，严重威胁粮食安全。2020年9月我国将小麦条锈病列为一类小麦病害防治对象，进行统防统治。小麦条锈病是全球流行性病害。20世纪40年代至今，小麦条锈菌新小种不断出现，导致我国8批次的主栽小麦品种丧失抗病性，造成病害全国大流行，最严重的年份小麦减产60亿公斤。

                                                  目前全球小麦生产受到小麦条锈病的潜在威胁，培育抗病优良品种是控制小麦条锈病最理想的措施，但是当前小麦抗病育种领域缺少综合性全景图，构建小麦抗条锈病位点全景图能够为小麦抗病育种提供参考。我们使用具有丰富多样性的 2191 份小麦材料基因型和条锈病表型数据，通过 GWAS 分析确定了 431 个与小麦条锈病显著关联的 QTL 位点，统计每个 QTL 对应的具体信息，并计算 QTL 在不同育种群和不同年代的频率变化。我们基于云计算框架开发了交互式在线全景图网站（LWDR），向全球小麦育种研究人员免费开放，提供小麦抗条锈病位点查询、分析、可视化等功能。

                                                  开源框架与协议

                                                  LWDR已经成功部署在云服务器，向全球用户提供便捷式服务。另外提供Docker、APP等多种本地部署方式，全部代码提交至Github开源，您可以根据需要进行修改，定制化自己的全景图。

                                                  开发者团队

                                                  西北农林科技大学、中科院遗传所、崖州湾国家实验室

                                                  3、技术框架

                                                  • 使用R语言shiny作为前端开发框架，Bootstrap组件库提供页面元素，使用Module模块化设计子功能
                                                  • 使用echarts4r、JavaScript、plotly等框架进行数据交互可视化，使用reactable、DT等进行数据表管理控制
                                                  • 使用tidyverse系列工具进行后端数据处理，vroom提供文件高速读写支持，parallel框架并行多线程计算
                                                  • 使用openxlsx、vcfR、bcftools、vcftools等工具进行文件解析，提供VCF、Xlsx等文件属性支持
                                                  • 使用RDS作为数据库储存格式，初始状态全局变量热启动加载进内存，大文件采用容器挂载方式使用
                                                  • 使用Docker容器技术进行整体开发，基于Ubtuntu20系统进行部署，由1Panel提供容器管理支持
                                                  • 使用阿里云分布式储存系统对网站静态资源进行结构化储存，加快响应速度，小文件使用OBS储存桶管理
                                                  • 使用华为云弹性云服务器ECS作为服务器端，提供公网权限和域名访问通道
                                                  • 使用shinyproxy进行流量代理与负载均衡，控制多用户并发使用场景，容器编排实现多项目并行

                                                  网站无响应，背景颜色变为灰色？

                                                  # VCF input fix function
                                                  get_vcf_df <- function(myfile = "www/download/Example_GeneType.filter.vcf"){
                                                      df_vcf <- read.vcfR(myfile)
                                                      df_out <- cbind(df_vcf@fix,df_vcf@gt) %>% as_tibble()
                                                      sample_list <- colnames(df_out)[-c(1:10)]
                                                      SNP_list <- df_out$ID
                                                      
                                                      df_table=df_out[c(1:30),c(1:8)]
                                                      
                                                      return(list(
                                                          df_out=df_out,
                                                          df_table=df_table,
                                                          sample_list=sample_list,
                                                          SNP_list=SNP_list,
                                                          info=paste0("Sample Number:",length(sample_list)," Marker Number:",length(SNP_list))
                                                      ))
                                                  }
